Guess Who game made with CognitiveJS.

Before making this project, I wanted to replicate game where you would answer questions to find your desired characters.

But because of the immense amount of data input needed for that, instead of making an Akinator copy, I've made a Guess Who game using the same algorithm that I've built for the Akinator copy.

For those who don't know, Guess Who is a game in which you try to find the opponent's character before the opponent finds your character by asking questions related to the set of 24 characters. All of the characters have distinctive features that could be identified, so you should ask questions and eleminate the ones who doesn't satisfy the criteria of your opponent's character.

The project itself is actually very extendable and could've added a lot of features, for example I've used a spritesheet to display the characters but it could've been done it dynamically, in which the name and the appearance of the characters generated procedurally. But I didn't bother adding these features mainly because it would take a lot of time and I'm not capable of illustrating or drawing in general.

The project is front-end only, so you can play the game directly inside your browser. Click the link in order to the game.
