Hotel reservation system made with B4X. Done it in only 2 weeks including the learning phase of the language.

This is the first project that I've done with . The reason why I didn't make this project with a more common language like C# is purely because of the need of learning a new language.

And even though I had relatively a short time using this language, I'm glad I have done this project with B4X because it is way better than it looks, a bit different than C# but it is simply awesome. Let me quickly explain the features that I like;

  • B4X, more specifically the desktop version called B4J is Java based so it can compile to every platform that supports Java.
  • The syntax is based on BASIC, if you are familiar with Visual Basic you'll feel right at home.
  • Design system is different than C#, on C# each form layout has its own corresponding code module. But on B4X, the layouts and code modules are completely seperate entities meaning that you can use the same layout for multiple code modules or the other way around.
  • IDE of the B4X is lightweight (especially compared to VS) and simple.

I'm looking forward to use B4X for my future projects.
