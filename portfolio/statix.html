A project for analyzing traffic of a website.

Point of this project is not just reporting web traffic but also using the visitor's data on various different internal projects that needs visitor's data, for example a contact form.

How does it work?

On the client-side of every page there will be AJAX request to a page called "tracker". Tracker gathers the visitor's credentials and generate a visitor ID based on the integrity of the credentials and save it to the database.

And after that, with the visitor ID, we can acquire visitor's data where we need it.
