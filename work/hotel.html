Hotel reservation system made with B4X. Done it in only 2 weeks including the learning phase of the language.


GitHub: Source


This is the first project that I've done with B4X . There are couple of reasons why I chose B4X instead of a more common language like C#, first of all I wanted to learn a new language, secondly there was no language restrictions for the school project and lastly it looked really promising.


I'm glad I have done this project with B4X because it is way better than it looks, a bit different than C# but it is simply awesome. Let me quickly explain the features that I like;


  • B4X, more specifically the desktop version called B4J is Java based so it can compile to every platform that Java supports.
  • The syntax is based on BASIC, if you are familiar with Visual Basic you'll feel right at home.
  • Design system is different than C#, on C# each form layout has its own corresponding code module. But on B4X, the layouts and code modules are completely seperate entities meaning that you can use the same layout for multiple code modules or the other way around.
  • IDE of the B4X is lightweight, simple and personally I think it's better than Visual Studio.
