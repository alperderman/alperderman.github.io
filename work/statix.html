An ongoing project for analyzing web traffic.


GitHub: Source


Point of this project is not just reporting web traffic but also using this visitor's data on various different internal projects that needs visitor's data for example contact form.

How does it going to work?


The project is going to be AJAX based so on the client-side of every page there will be AJAX request to a page called "Tracker". Tracker will gather visitor's credentials and generate a visitor ID based on the integrity of the credentials and save it to the database.


And after that we can use this visitor ID on places where we need visitor's credentials. It is easy as it is sounds like but the hard part is making a full fledged dashboard that procceses this data and visualizes it.

Future Goals


  • Make a dashboard
  • Make a real-time tracking system
  • Make an API handler to retrieve the data externally
  • Make a mobile app of the dashboard using the API handler
