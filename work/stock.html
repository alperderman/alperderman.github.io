Stock management software made with C#. It was my school project that I've done it in only 5 days which was just a day before due.


GitHub: Source


Apart from the usual features, I have added a permission feature so that only certain users control certain part of the system for instance if you are a salesman on the system you can only sell products, you cannot do anything else like buying or changing the price of a product.


There are total of 5 job titles and each has it's own unique role on the system.
